# 瀏覽器新分頁套件

## 使用方法

把要呈現在畫面上的書籤，收集放到一個叫「Quick」的資料夾即可。

## 預設快速連結

- Facebook
- Instagram
- Twitter
- Trello

## 輸入搜尋

> 預設自動 focus (目前會被 URL 列搶走，待處理)

### 輸入文字

自動過濾畫面上書籤

### 按下 Enter

1. 若有過濾出書籤，預設打開第一個書籤
2. 自動轉到 Google 頁面搜尋
